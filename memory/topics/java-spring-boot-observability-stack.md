

## 技能与技术栈约定

- **核心领域**：Java 技术栈（Spring Boot 3 微服务）与可观测性/运维协同场景深度优化。
- **已启用技能**：
  - 🕒 时间查询：返回用户本地时区当前时间。
  - 📚 内部文档检索：支持 RAG 实现规范、Spring Boot 生产配置模板、JVM 调优 checklist、Milvus 向量索引策略等知识库精准搜索。
  - 🚨 Prometheus 告警查询：拉取 firing 状态告警，含标签、注解、触发值，用于根因定位。
  - 📜 日志分析：覆盖 `application-logs` / `system-metrics` / `database-slow-query` / `system-events` 四类主题，支持 Lucene 语法深度查询。
  - 🌐 网页抓取：安全获取公开网页或 API（如 Spring 官方文档、GitHub README、Maven 信息）。
  - 🔍 公共网络搜索：提供 Java/云原生/向量数据库等技术概念的权威摘要（标注“公开信息摘要”，不替代官方文档）。
  - 🌤️ 天气与地理信息：支持城市天气、地理编码（经纬度、行政区划），用于 GIS 集成或地域模拟测试。
- **长期聚焦方向**（executionId: exec_1e1a0ade63a9491fa74476f5e23c437b）：
  - 当前项目架构：Spring Boot 3 + MyBatis-Plus + Milvus 向量引擎 + 静态前端。
  - Java 生产最佳实践：内存泄漏防控、GC 日志解读、线程池监控、Actuator 健康检查定制。
  - 工具链协同逻辑：「告警 → 日志 → JVM 指标」闭环排查路径。
